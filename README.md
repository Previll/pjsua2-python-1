# pjsua2-python

Binary wheels for Python bindings to PJSUA2 (PJSIP 2.17).

## Platforms

- Linux x86_64 (manylinux2014)
- Linux i686 (manylinux2014)
- Linux ARM64/aarch64 (manylinux2014)
- Linux ppc64le (manylinux2014)
- Linux s390x (manylinux2014)
- Linux armv7l (manylinux_2_31)
- Linux riscv64 (manylinux_2_39)
- Windows AMD64

## Usage

Install from a wheel:

```bash
pip install pjsua2-python==2.17.0
